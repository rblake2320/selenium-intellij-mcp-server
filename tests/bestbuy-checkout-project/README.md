# Best Buy PS5 Guest Checkout — Selenium Tests

Automated Selenium WebDriver tests that navigate bestbuy.com, search for a PS5, add it to cart, and proceed to guest checkout — **stopping before placing any order**.

## Prerequisites

- **JDK 17+** (set `JAVA_HOME`)
- **Maven 3.8+**
- **Chrome browser** installed
- ChromeDriver is auto-managed by WebDriverManager

## Quick Start

```bash
# Clone and navigate
git clone https://github.com/rblake2320/selenium-intellij-mcp-server.git
cd selenium-intellij-mcp-server/tests/bestbuy-checkout-project

# Run all tests
mvn test

# Run a specific test
mvn test -Dtest=BestBuyPS5GuestCheckoutTest
mvn test -Dtest=BestBuyPS5RecordedTest
```

## Tests Included

| Test | Description |
|------|-------------|
| `BestBuyPS5GuestCheckoutTest` | Hand-crafted, robust test with multi-selector fallbacks, smart product filtering, popup handling, and screenshot capture |
| `BestBuyPS5RecordedTest` | Lighter test generated by the MCP recording service — linear flow with single selectors |

## Open in IntelliJ IDEA

1. **File → Open** → select the `bestbuy-checkout-project` folder
2. IntelliJ detects the `pom.xml` and imports as Maven project
3. Wait for indexing + dependency download
4. Right-click any test class → **Run**
